Helpful Linux command and what they do


# sudo find / -type f -size +4G
Find files greater than 4 gigabytes from the root folder
. G -> Gigabytes
. M -> Megabytes
. K-> for kilobytes
. b-> for bytes

# sudo find / -type f -size -4G
Find files lesser than 4 gigabytes from the root folder
